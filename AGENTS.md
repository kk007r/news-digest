# AGENTS.md

このフォルダでは、個人用のニュースまとめツールを小さく安全に育てる。

## 作業ルール

- 変更前に作業計画を出す。
- 1回の変更は小さくする。
- 追加課金が発生するサービスは使わない。
- OpenAI APIは使わない。
- LINE連携は使わない。
- LINE Messaging API、LINE Notifyは使わない。
- APIキーやトークンが必要な構成にしない。
- 有料API、有料クラウド、ログイン必須サイトのスクレイピングは使わない。
- ニュース本文の全文転載はしない。
- RSSやAtomなど公開フィードを優先する。
- READMEに実行方法と確認方法を書く。
- 初心者でも保守しやすいコードにする。
- 不要なライブラリを増やさない。
- まず手動実行、その後GitHub Actions対応の順に進める。
